Introduccin

WYZTracker es una herramienta desarrollada por Augusto Ruiz, de Retroworks, para componer msica para el WYZ Player, un reproductor que puede utilizarse en equipos con CPU Z80 y chip de sonido AY.

Un primer vistazo a la aplicacin

A continuacin podemos ver una captura de pantalla de la ventana que aparece cuando arrancamos la aplicacin por primera vez:


Se puede apreciar la barra de men, la barra de herramientas debajo, y debajo de sta lo siguiente (de izquierda a derecha):

  • Editor de orden de patrones
  • Editor de propiedades de cancin
  • Ajustes de envolventes (para el instrumento Sawtooth)
  • Editor de instrumentos y efectos
Debajo de esta zona se puede ver el editor de patrones, que permite editar el patrn seleccionado en el editor de orden de patrones.

El men archivo tiene las siguientes opciones:

Que permite hacer lo siguiente:

Barra de herramientas


La barra de herramientas tiene una serie de botones que permiten hacer lo siguiente:

 Crear un archivo nuevo.
 Abrir un archivo existente.
 Guarda el archivo actual.
 Reproducir la cancin actual desde el principio.
 Reproducir la cancin actual desde el patrn actual.
 Reproducir el patrn actual, repitindole una y otra vez.
 Parar la reproduccin. Si pulsas estando parada la reproduccin, te llevar al inicio de la cancin.
 Aade un nuevo patrn a la lista de patrones. El nuevo patrn se aade justo detrs del patrn seleccionado, y el Id del nuevo patrn es el mismo que el del patrn seleccionado.
 Elimina el patrn seleccionado de la lista de patrones. Ojo, esto no borra el patrn, simplemente lo quita de la lista de patrones.
 Crea una copia exacta del patrn seleccionado, y aade esta copia a la lista de patrones, justo debajo del seleccionado.
 Importa un fichero .MUS. No est implementado an :(
 Exporta la cancin actual a archivos .MUS y .ASM.

Hay otros controles en la barra de herramientas:

Octava Base: Octava a utilizar para las notas que se aaden utilizando el piano virtual en el editor de patrones. Se puede incrementar con F2, y decrementar con F3.
Incremento Automtico: Nmero de espacios que avanza el cursor cuando se aade una nueva nota utilizando el piano virtual en el editor de patrones. Se puede establecer a cero si queremos que el cursor se quede en el mismo sitio. Se puede incrementar con F4, y decrementar con F5.
Resaltar cada: Establece el intervalo de notas a las que se le aplicar un resaltado en el editor de patrones.
Stereo: Establece el modo estreo a utilizar en la cancin actual. Slo es vlido para escuchar las canciones o para la exportacin de audio, pero no afectar a la exportacin a MUS.

Por cierto, todos los botones tienen un tooltip que se mostrar si dejas el ratn quieto sobre el botn.

Editor de orden de patrones (selector de patrones)


El editor de orden de patrones permite aadir, quitar y ordenar los patrones de la cancin actual. Tambin es el sitio donde se pueden crear patrones nuevos, y especificar la longitud del patrn seleccionado. El patrn a editar se selecciona en este editor, pulsando en la lista de patrones.

La funcionalidad disponible es la siguiente:

  • Lista de patrones: Pulsa sobre cualquier patrn para seleccionarlo. Aparecer en el editor de patrones.
  • Flecha arriba (): Mueve el patrn seleccionado hacia arriba en la lista.
  • Flecha abajo (): Mueve el patrn seleccionado hacia abajo en la lista.
  • Ms (): Incrementa el ID del patrn que se encuentra en la posicion seleccionada. Si no existe el patrn con el ID resultante, se crea un nuevo patrn con dicho ID, de la longitud establecida actualmente. Esto no cambia el patrn seleccionado, lo que hace es indicar que en la posicin actual se encuentra el patrn con ID = ID + 1.
  • Menos (): Decrementa el ID del patrn que se encuentra en la posicin seleccionada. Al igual que en el caso anterior, esto no cambia el patrn actualmente seleccionado. Simplemente indica que en la posicin actual se encuentra el patrn con ID = ID - 1.
  • Longitud: Establece la longitud del patrn seleccionado. Cuidado: si la nueva longitud es menor que la anterior, perders los datos de las notas que ya no formen parte del patrn!

Editor de propiedades de cancin


El editor de propiedades permite establecer propiedades generales de la cancin, como pueden ser el ttulo, el tempo, el nmero de canales...

Los datos que se puedne especificar son los siguientes:

  • Nombre: Ttulo de la cancin
  • Canales: Nmero de canales (1-8)
  • Tempo: Tempo de la cancin. Cuanto mayor el nmero, ms lento es el tempo. Representa el nmero de interrupciones que dura cada posicin en el patrn. Las interrupciones suceden 50 veces por segundo en ordenadores PAL, y 60 en ordenadores NTSC.
  • Bucle: Indica si la cancin ha de empezar su reproduccin desde el principio una vez que la cancin termina o no.
  • Mquina: Indica la frecuencia de emulacin del chip AY. MSX/Spectrum (1.77MHz) o CPC (1 MHz). De esta manera se puede comprobar cmo sonar la cancin actual en los diferentes tipos de mquinas (si bien, se pueden modificar las tablas de frecuencias para ajustar a tu gusto cmo suena la cancin).

Editor de ajustes de envolvente


El editor de ajustes de envolvente se utiliza para establecer los parmetros que se aplicarn al aadir notas que utilicen el instrumento Sawtooth. Slo aplica a las notas aadidas, no modificar los valores de las notas existentes (a no ser que se aada una nota nueva que reemplace una existente).


Los valores son los mismos que se estableceran en el chip AY para controlar las envolventes, por lo que recomiendo probar diferentes combinaciones para encontrar el sonido buscado. Para ello basta con seleccionar el instrumento Sawtooth, modificar los valores y en el editor de patrones pulsar cualquier tecla del piano virtual. Y por favor, prueba tambin con el check de Frecuencias Activas seleccionado!! ;)

Selector de instrumentos


Lista de instrumentos que se han definido para la cancin actual. Se puede cambiar el instrumento seleccionado, que es el que se usar cuando se pulsen teclas del piano virtual. Si pulsamos el botn derecho sobre un instrumento, podremos exportarlo individualmente. Para exportar todos los instrumentos, utiliza la barra de men o el botn exportar que se encuentra en la barra de herramientas debajo de la lista de instrumentos.


Haciendo doble click sobre un instrumento se mostrar el editor de instrumentos.

Debajo de la lista hay una barra de herramientas que permite crear instrumentos nuevos, borrar el instrumento seleccionado, exportar todos los instrumentos, mostrar el editor de instrumentos para el instrumento seleccionado, y mostrar el nombre del instrumento actual.

Se puede cambiar de instrumento seleccionado de forma rpida pulsando F6 y/o F7.

Selector de efectos


Lista de efectos definidos para la cancin actual. Haciendo click con el botn derecho sobre un efecto permite exportar el efecto seleccionado. Para exportar todos los efectos, utiliza la barra de men, o el botn exportar ubicado en la barra de mens que se encuentra debajo de la lista de efectos.


Haciendo doble click sobre un efecto muestra el editor de efectos.

Debajo de la lista hay una barra de herramientas que permite crear efectos nuevos, borrar el efecto seleccionado, exportar todos los efectos, mostrar el editor de efectos para el efecto seleccionado, y mostrar el nombre del efecto actual.

Editor de patrones


Por fin! El editor de patrones es donde indicas las notas que se van a reproducir para el patrn actual, con qu instrumento se deber tocar, y durante cunto tiempo. El piano virtual te permitir hacerlo de forma sencilla. Cada vez que pulses una tecla con el editor de patrones, si esa tecla pertenece al piano virtual, se aadir una nota al editor en la posicin seleccionada.

Trabajando con el piano virtual


El piano virtual tiene dos octavas. La octava inferior compienza en la tecla Z, que corresponde a un Do, y todas las teclas en dicha fila son Re, Mi, Fa, Sol, etc. La siguiente fila de teclas es donde se ubican los sostenidos. Por lo tanto, existen teclas que no tienen nota asociada. Por ejemplo, Do sostenido est en la tecla S. El valor de la octava a aplicar es el valor seleccionado en el selector "Octava Base" de la barra de herramientas principal.

Si la posicin seleccionada en el patrn es del canal de efectos, el piano virtual trabaja de forma diferente. Se utilizan nicamente las teclas 1,2,3... del teclado principal para seleccionar el efecto a utilizar.

Editor de instrumentos

Aqu puedes editar, aadir o eliminar instrumentos. Tiene la siguiente pinta:


A la izquierda se puede ver el selector de instrumentos, similar al que aparece en la pantalla principal. Se puede pulsar en cualquier instrumento para ver su definicin a la derecha. Debajo del selector hay una barra de herramientas donde se puede:

 Crear un instrumento nuevo.
 Eliminar el instrumento seleccionado (excepto el instrumento Sawtooth)
 Importar instrumentos de un fichero .ins o .wyz.
 Exportar todos los instrumentos.

Puedes exportar un instrumento por separado haciendo click con el botn derecho sobre el instrumento, y seleccionando "Exportar instrumento...".

Definicin de instrumentos

Un instrumento se define mediante los siguientes datos:
  • Id: Identificador del instrumento, un valor numrico utilizado en el las notas de los patrones,
  • Nombre: Nombre del instrumento.
  • Longitud: Longitud del instrumento.
  • Bucle: Indica si el instrumento tiene un bucle, y de ser as, en qu posicin comienza dicho bucle.
  • Volmenes.
  • Modificadores de octava.
  • Modificadores de tono.

Volmenes y modificadores

Se pueden establecer los volmenes haciendo click en las columnas impares del editor de instrumentos, y arrastrando. El checkbox "Bloquear" se puede marcar para que el cambio del valor slo se produzca en la columna sobre la que se hizo click inicialmente.

Los modificadores pueden ser:
  • Modificadores de octava: Hacer click con el botn derecho sobre el volumen, y selecciona el modificador de octava a aplicar utilizando el men contextual que aparece.
  • Modificador de tono: Modificador acumulativo. Hacer click con el botn izquierdo del ratn sobre una columna par. Se aplica al volumen anterior.
  • Modificador de nota: Hacer click con el botn derecho en una columna par.

Pruebas

El instrumento actual se puede probar pulsando el botn "Probar", el cual se mantendr pulsado. A partir de ese momento, se puede utilizar el piano virtual para probar el sonido del instrumento actual. Pulsando el botn de nuevo, la prueba terminar. Tambin puedes probar el instrumento actual si la cancin se est reproduciendo, ya que los cambios se aplicarn inmediatamente sobre la cancin.

Editor de efectos



A la izquierda se puede ver el selector de efectos, similar al que aparece en la pantalla principal. Se puede pulsar en cualquier efecto para ver su definicin a la derecha. Debajo del selector hay una barra de herramientas donde se puede:

 Crear un efecto nuevo.
 Eliminar el efecto seleccionado.
 Importar efectos de un fichero .fx o .wyz.
 Exportar todos los efectos.

Puedes exportar un efecto por separado haciendo click con el botn derecho sobre el efecto, y seleccionando "Exportar efecto...".

Definiendo efectos

Un efecto est definido por los siguientes datos:
  • Id: Identificador del efecto, un valor numrico utilizado por las notas del canal FX.
  • Nombre: Nombre del efecto.
  • Longitud: Longitud del efecto.
  • Volmenes.
  • Niveles de ruido.
  • Frecuencias.

Estableciendo los niveles

Se pueden establecer los niveles (volumen, ruido, frecuencia) haciendo click con el botn izquierdo y arrastrando. El checkbox "Bloquear" permite restringir el cambio de valor al elemento sobre el que se hace click. Los niveles estn codificados por color:
  • Volumen: Barras verdes
  • Ruido: Barras rojas
  • Frecuencia: Barras amarillas

Probando el efecto

Pulsa el botn Probar, y se mantendr pulsado. Mientras est pulsado, podrs usar el piano virtual para probar los efectos. Pulsando de nuevo el botn Probar terminar la prueba.

Editor de frecuencias


Editor donde se puede especificar la frecuencia a utilizar por nota. Se puede seleccionar:
  • Una tabla "estndar" para MSX/Spectrum
  • Una tabla "estndar" para CPC
  • Una tabla parametrizada: Cambia el valor del control para establecer frecuencias mayores o menores.
  • Tabla personalizada: Para especificar para cada nota de cada octava el valor de frecuencia a utilizar.
Se puede guardar y recuperar la tabla para utilizarla en diferentes canciones.


Editor de arpegios

Permite rellenar de forma sencilla un canal de un patrn, con una serie de herramientas visuales.

Se pueden especificar una serie de parmetros que controlan la nota raz y cuntas octavas quieres utilizar en tu arpegio. Tambin el patrn y canal sobre el que las notas se escribirn.

Los parmetros son:
  • Instrumento: Instrumento a utilizar en el arpegio
  • Octava inicial: Octava de la nota raz.
  • Nota inicial: Nota raz.
  • Longitud: Numero de notas del arpegio.
  • Resaltar por escala: Permite ver en la parte inferior las notas de la escala seleccionada resaltadas.
  • Patrn destino: El patrn donde las notas se escribirn cuando pulses "Aplicar". Se puede seleccionar "Nuevo patrn", para crear un patrn nuevo y aadirlo a la lista de patrones.
  • Longitud del patrn: Solo disponible si el patrn destino es un nuevo patrn. Establece la longitud de dicho nuevo patrn.
  • Canal: Canal destino sobre el que se escribirn las notas.
  • Mx. Octavas: Nmero de octavas que se muestran alrededor de la nota raz.


En la parte inferior del editor se puede ver una rejilla donde se pueden especificar las notas del arpegio. Las notas se introducen haciendo click con el botn izquierdo en la rejilla con el botn izquierdo. Se muestran en color rojo. Si se pulsa con el botn derecho sobre una nota existente, se borrar la nota. Si no existe una nota, se insertar un silencio (pintado en naranja).

Los botones disponibles son:

  • Aplicar: Copia las notas del arpegio al canal y patrn seleccionados.
  • Cargar: Carga una configuracin de arpegio previamente guardada.
  • Guardar: Guarda la configuracin actual de arpegio.
  • Generar aleatorio: Crea un conjunto de notas aleatorias en el arpegio actual, borrando las actuales. Si hay seleccionada alguna escala, solo se utilizarn notas de dicha escala.
  • Probar: Reproduce el arpegio actual.
  • Bloquear: Como en el editor de instrumentos y de efectos, bloquear permite evitar modificar valores de otras columnas cuando arrastras con el ratn.
